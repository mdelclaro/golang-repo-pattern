## ⚙️ Installation

Make sure docker daemon is running. To install/build the dependencies:

```bash
 make build-dependencies
```

## ⚡️ Running The API

```bash
 make run
```

## 🧪 Test All

```bash
 make test
```

## 📖 Documentation
 Copy the file `openapi.json` and paste it into https://editor.swagger.io/ to visualize it.
